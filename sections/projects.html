Spotify Data Analysis

This project showcases end-to-end data processing and analysis using data from the Spotify API. It includes ETL pipelines built in Azure Databricks, leveraging Python, PySpark, and SparkSQL for data transformation and analysis. Final insights are presented through interactive Power BI dashboards. The analysis focuses on user playlists, with input driven by provided playlist IDs.

Phone Book

A C# application built with Entity Framework Core and SQL Server that enables full CRUD operations on contact data. Users can manage contacts with details like name, phone number, email, and category (e.g., Family, Work), and filter them by category. The app also features integrated communication tools, allowing users to send emails and SMS messages directly from the app using Twilio.

Drinks Menu

A C# application that connects to an external API to display a restaurant’s drinks menu. Users can browse by category and view detailed drink information through a clean, user-friendly interface. The app filters out empty fields to ensure only complete, relevant data is shown.

Shifts Logger

The Shifts Logger Application is a powerful C# .NET application designed to help users track their work shifts efficiently. It leverages Entity Framework Core for database operations and uses SQL Server as the backend database. The application allows to create an account for users who do not exist in the system, ensuring they can start managing their shifts seamlessly.

Exercise Tracker

A simple C# console application to help users track their training and exercise sessions. The app supports two types of exercises: Weight and Cardio, based on two ORMs - EF Core and Dapper, depending on selected exercise type.

Sports Results

This is a C# application that scrapes basketball game results and sends them via email. It leverages the HtmlAgilityPack library for HTML parsing and uses Gmail's SMTP service to send emails.

Excel Reader

Excel Reader is a C# console application designed for efficient Excel file processing and database management. The app offers an interactive command-line UI using Spectre.Console, leverages EPPlus for Excel operations, and utilizes Entity Framework Core (EFCore) to communicate with a database.

Flashcards

A C# console application for managing and studying flashcards, built with a focus on simplicity and functionality. This application allows you to create, organize, and study flashcards grouped into stacks. You can track your study sessions and analyze your progress over time.
